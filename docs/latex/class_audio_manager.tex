The class for playing sound in the game. It is a singleton object which is persisted between scenes via Dont\+Destroy\+On\+Load. The sound is divided into two cattegories\+: sounds and music. Sounds or sound effects are short clips such as footsteps, sword crash, etc. Music is played in the background.
